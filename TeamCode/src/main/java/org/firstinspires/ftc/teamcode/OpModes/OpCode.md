# DECODE 机器人游戏手柄按键功能

## Gamepad1（一操）按键功能

| 按键           | 功能                         |
| ------------ | -------------------------- |
| **A 按钮**     | 切换到等待状态（非攀爬模式）             |
| **B 按钮**     | 切换攀爬模式/等待状态                |
| **X 按钮**     | 释放时切换无头模式                  |
| **Y 按钮**     | 切换 ReadyToShoot 状态（进入射击模式） |
| **左保险杠**     | 进入吃球模式                     |
| **右保险杠**     | 进入吐球模式/停止吐球模式              |
| **左扳机**      | 按下超过 0.6 时自动对准到射击角度        |
| **右扳机**      | 按下时自动对准到吃球角度               |
| **左摇杆**      | 控制机器人前进/后退（Y 轴）和左右平移（X 轴）  |
| **右摇杆**      | 控制机器人旋转（X 轴）               |
| **方向键上**     | 导航到远侧射击点                   |
| **方向键下**     | 导航到近侧射击点                   |
| **方向键左**     | 调整射击速度（使用传统方法）             |
| **方向键右**     | 增加航向误差阈值                   |
| **Start 按钮** | 无特定功能                      |
| **Back 按钮**  | 无特定功能                      |

## Gamepad2（二操）按键功能

| 按键           | 功能                                          |
| ------------ | ------------------------------------------- |
| **A 按钮**     | 切换到等待状态（非攀爬模式）                              |
| **B 按钮**     | 切换 DRL 模式（RunDRL →NoDRL → FullDRL → RunDRL） |
| **X 按钮**     | 切换到等待状态并停止清扫器                               |
| **Y 按钮**     | 进入射击模式                                      |
| **左保险杠**     | 进入吃球模式/停止吃球模式                               |
| **右保险杠**     | 进入吐球模式/停止吐球模式                               |
| **左扳机**      | 直接控制电梯上升                                    |
| **右扳机**      | 直接控制电梯下降                                    |
| **左摇杆**      | 辅助控制机器人移动                                   |
| **右摇杆**      | 辅助控制机器人旋转                                   |
| **方向键上**     | 增加速度修正系数                                    |
| **方向键下**     | 减少速度修正系数                                    |
| **方向键左**     | 减少角度偏移                                      |
| **方向键右**     | 增加角度偏移                                      |
| **Start 按钮** | 在射击模式下控制清扫器和触发器                             |
| **Back 按钮**  | 无特定功能                                       |
| **左摇杆按钮**    | 导航到攀爬点（攀爬模式下）                               |
| **右摇杆按钮**    | 导航到攀爬点（攀爬模式下）                               |
